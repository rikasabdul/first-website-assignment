Horiseon Website

A website that showcases Horiseon's services in Search Engine Optimization (SEO), Online Reputation Management, and Social Media Marketing.
Changes Made

    Updated the CSS using Flexbox to improve the layout and alignment of the elements in the website.
    Fixed the bugs in the alignment of the images and text.
    Changed the float type to center for the images in the sections for SEO, Online Reputation Management, and Social Media Marketing.

Files and Folders

    index.html: The main HTML file that contains the structure and content of the website.
    assets/: A folder that contains all the assets used in the website, such as images and CSS files.
    assets/css/style.css: The CSS file that contains the styles used in the website.
    assets/images/: A folder that contains all the images used in the website.

How to run the website

To run the website, simply open the index.html file in a web browser. You can also run it by deploying the website on a web server.
